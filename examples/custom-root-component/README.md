# Custom root component

## How to use

```bash
git clone https://github.com/ream/ream.git
```

Install dependencies:

```bash
cd examples/custom-root-component
yarn
```

Run it:

```bash
# Start development server
yarn dev

# Start production server
yarn build && yarn start
```
